The operation add(a, b) means combining two numbers called a and b to find their total sum.

Here, a is the first number also called the augend. It is the number we start with.
Here, b is the second number also called the addend. It is the number we add to a.
The result of add(a, b) is called the sum. It represents the total quantity when a and b are combined.

a and b can each be any integer: positive, negative, or zero.
A positive integer is greater than zero. Examples: 1, 5, 42, 347.
A negative integer is less than zero. It is written with a minus sign in front. Examples: -1, -5, -42, -347.
Zero is neither positive nor negative. Zero written as 0.

Addition is commutative: add(a, b) gives the same result as add(b, a).
Addition is associative: add(add(a, b), c) gives the same result as add(a, add(b, c)).
The identity element of addition is 0: add(a, 0) equals a and add(0, a) equals a for any integer a.

When both a and b are positive, we add their digit columns from right to left using carry-over.
When both a and b are negative, we add their absolute values and attach a negative sign to the result.
When a and b have opposite signs, we subtract the smaller absolute value from the larger and keep the sign of the number with the larger absolute value.
When one of a or b is zero, the result is simply the other number.

Each digit position represents a place value.
The rightmost digit is the ones place with place value 1.
The second digit from the right is the tens place with place value 10.
The third digit from the right is the hundreds place with place value 100.
The fourth digit from the right is the thousands place with place value 1000.
Each position is 10 times the value of the position to its right.

When the sum of two digits in the same position is 10 or greater we cannot write a two-digit number in a single position. We write only the units digit of the sum in that position and carry the tens digit which is always 1 over to the next higher position. This is called carrying or carry-over.

For example if the ones digits sum to 13 we write 3 in the ones place and carry 1 to the tens place.
The carried 1 is then added to the sum of the tens digits before the same rule is applied again.
This process repeats for every digit position until there are no more digits and no remaining carry.


def add_unit(a, b):
    # a is a single digit: 0 <= a <= 9
    # b is a single digit: 0 <= b <= 9
    # both a and b are non-negative single digit integers for this function
    total = a + b
    # total is the raw sum of the two unit digits
    units = total % 10
    # units is the digit we write in the current column which is total mod 10
    carry = total // 10
    # carry is 1 if total >= 10 otherwise 0 which is total integer-divided by 10
    return units, carry
    # returns a tuple of the digit to write and the carry to pass to the next column

# add_unit(3, 4) gives total=7, units=7, carry=0 so result is (7, 0) with no carry
# add_unit(7, 5) gives total=12, units=2, carry=1 so result is (2, 1) with carry of 1
# add_unit(9, 9) gives total=18, units=8, carry=1 so result is (8, 1) with carry of 1


def add(a, b):
    # a is any integer positive negative or zero
    # b is any integer positive negative or zero

    if a == 0:
        return b
        # identity: adding zero to b gives b

    if b == 0:
        return a
        # identity: adding zero to a gives a

    if a > 0 and b > 0:
        # both positive: add digit by digit with carry from right to left
        result_digits = []
        carry = 0
        # carry starts at 0 because there is no carry before the first column
        while a > 0 or b > 0 or carry > 0:
            # loop continues as long as there are digits left in a or b or a leftover carry
            digit_a = a % 10
            # digit_a is the current rightmost digit of a
            digit_b = b % 10
            # digit_b is the current rightmost digit of b
            column_sum = digit_a + digit_b + carry
            # column_sum is the total for this column: digit of a plus digit of b plus carry
            carry = column_sum // 10
            # new carry is 1 if column_sum >= 10 else 0
            result_digits.append(column_sum % 10)
            # write only the units digit of column_sum
            a = a // 10
            # remove the rightmost digit of a
            b = b // 10
            # remove the rightmost digit of b
        result_digits.reverse()
        # result_digits is currently ones-first so reverse to get correct order
        result = 0
        for digit in result_digits:
            result = result * 10 + digit
        # reconstruct the integer from the list of digits
        return result

    if a < 0 and b < 0:
        # both negative: add their absolute values then negate
        # add(-3, -5) is the same as -(add(3, 5)) which equals -8
        return -add(-a, -b)
        # -a and -b are both positive so this recurses into the positive branch above

    if a < 0 and b > 0:
        # opposite signs: subtract smaller absolute value from larger
        abs_a = -a
        # abs_a is the absolute value of a
        abs_b = b
        # abs_b is already positive
        if abs_a == abs_b:
            return 0
            # equal magnitudes with opposite signs cancel out to zero
        if abs_b > abs_a:
            return subtract(abs_b, abs_a)
            # b has larger magnitude so result is positive: b minus the magnitude of a
        else:
            return -subtract(abs_a, abs_b)
            # a has larger magnitude so result is negative: negate (magnitude of a minus b)

    if a > 0 and b < 0:
        # opposite signs: same logic as above but roles of a and b are swapped
        abs_a = a
        abs_b = -b
        if abs_a == abs_b:
            return 0
        if abs_a > abs_b:
            return subtract(abs_a, abs_b)
        else:
            return -subtract(abs_b, abs_a)


def subtract(larger, smaller):
    # helper used when adding numbers of opposite signs
    # larger and smaller are both non-negative integers and larger >= smaller
    # returns larger minus smaller as a non-negative integer
    result_digits = []
    borrow = 0
    # borrow starts at 0
    a = larger
    b = smaller
    while a > 0 or b > 0:
        digit_a = a % 10
        # rightmost digit of larger
        digit_b = b % 10
        # rightmost digit of smaller
        diff = digit_a - digit_b - borrow
        # diff is the subtraction at this column including any borrow from previous column
        if diff < 0:
            diff = diff + 10
            # if diff is negative we borrow 10 from the next column
            borrow = 1
            # record that we borrowed
        else:
            borrow = 0
            # no borrow needed
        result_digits.append(diff)
        a = a // 10
        b = b // 10
    while result_digits and result_digits[-1] == 0:
        result_digits.pop()
        # remove leading zeros from the reversed list
    if not result_digits:
        return 0
    result_digits.reverse()
    result = 0
    for digit in result_digits:
        result = result * 10 + digit
    return result
